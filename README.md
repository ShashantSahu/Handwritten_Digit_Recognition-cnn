# 🧠 Handwritten Digit Recognition using CNN

This project is a Deep Learning-based Handwritten Digit Recognition system built using TensorFlow and Keras. It classifies handwritten digits (0–9) using a Convolutional Neural Network (CNN) trained on the MNIST dataset.

---

## 🚀 Project Overview
The goal of this project is to correctly identify handwritten digits from images using a trained deep learning model. The system uses Convolutional Neural Networks (CNN) for better accuracy and feature extraction.

---

## 📊 Dataset
- MNIST Dataset
- 70,000 grayscale images
- Image size: 28x28 pixels
- 10 classes (digits 0–9)

---

## 🧠 Model Architecture
- Convolutional Neural Network (CNN)
- Conv2D layers
- MaxPooling layers
- Flatten layer
- Dense layers
- Softmax output layer

---

## ⚙️ Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

---

## 🎯 Features
- Digit classification (0–9)
- Image preprocessing and normalization
- CNN-based deep learning model
- Real-time prediction on test data
- High accuracy (~97–99%)

---

## 📈 Results
- Training Accuracy: ~99%
- Test Accuracy: ~97–99%
